\# Age Category Prediction



Проект по построению модели многоклассовой классификации для определения возрастной категории пользователя на основе цифрового поведения.



\## Задача



Построить модель машинного обучения, предсказывающую возрастную категорию пользователя:



0 - <18  

1 - 18–25  

2 - 26–40  

3 - 41–55  

4 - 56+



\## Используемые данные



\- Логи посещений сайтов

\- Рекламная активность (CTR)

\- Глубина просмотра

\- Тип устройства

\- Использование облачных сервисов



\## Модель



\- Logistic Regression (multinomial)

\- class\_weight="balanced"

\- C = 5

\- Macro F1 ≈ 0.52



\## Стек



\- Python

\- pandas

\- scikit-learn

\- seaborn

\- matplotlib



\## Результат



Модель демонстрирует стабильное качество и может использоваться для сегментации аудитории по возрастным категориям.
